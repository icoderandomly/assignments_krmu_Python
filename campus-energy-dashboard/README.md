# campus-energy-dashboard-Harsh_Singh

This repository contains a complete solution for the Campus Energy-Use Dashboard capstone assignment. It includes:
- Data ingestion (reads CSVs from /data/)
- Aggregation logic (daily, weekly, building-wise)
- Object-oriented design (Building, MeterReading, BuildingManager)
- Visualizations (trend, bar, scatter) stitched into `output/dashboard.png`
- Output files in `/output/` including `cleaned_energy_data.csv`, `building_summary.csv`, and `summary.txt`

**How to run**

2. Run: `python src/main.py`
3. Outputs will be in `output/` directory.
